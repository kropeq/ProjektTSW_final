ProjektTSW_final
================

Projekt na zaliczenie

Sposób uruchomienia:
====================
1. Pobrać całość z githuba.
2. W folderze z projektem wpisać npm install
3. By uruchomić npm start
4. W przeglądarce działamy przez localhost:3000.

Wykorzystane Technologie:
=========================

1. Express
2. Socket.io
3. Moment
4. MongoDB
5. Body parser
6. Cookie parser
7. Session
8. EJS
9. LESS
